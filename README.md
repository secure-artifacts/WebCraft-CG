# WebCraft 网页工坊

Chrome MV3 extension for webpage capture, translation, screen recording, YouTube thumbnail tools, and a lightweight collection canvas.

## Features

- Screenshot current viewport and long pages
- Region screen recording
- Google Translate and DeepL translation with fallback
- Floating translator panel on webpages
- YouTube thumbnail download and recommendation filtering
- Collection canvas for selected text, links, and images
- Import/export settings and changelog page

## Install Locally

1. Open `chrome://extensions/`.
2. Enable Developer mode.
3. Click **Load unpacked**.
4. Select this folder.

## Privacy

API keys are stored in local browser extension storage. Do not share exported settings if they contain private keys.
